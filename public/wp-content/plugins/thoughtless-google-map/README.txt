TO USE THE PLUGIN:
In root, make a file called: credentials.php

in this file define three constants:
define('THOUGHTLESS_GOOGLE_MAP_API_KEY', '');   --  Type in a valid api key to google
define('THOUGHTLESS_GOOGLE_MAP_USERNAME', '');  --  Type in a valid admin username to your wordpress site
define('THOUGHTLESS_GOOGLE_MAP_PASSWORD', '');  --  Type in a valid admin password to your wordpress site
